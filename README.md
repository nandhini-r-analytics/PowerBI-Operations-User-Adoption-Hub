# Enterprise Operations & User Adoption Hub (Power BI & SQL)

## 📌 Project Overview
A comprehensive look into transforming static, legacy SharePoint reports into an automated, end-to-end Power BI reporting ecosystem to track core operational KPIs and user adoption trends.

## 🏢 Business Problem
The operations team relied heavily on static reporting workflows, causing massive delays in tracking daily key performance indicators and identifying revenue leakages.

## 🛠️ Technical Solution & Toolkit
- **Data Extraction:** Optimized complex SQL queries to extract data from high-volume transaction databases.
- **Data Modeling:** Designed a robust star-schema data model mapping workflows directly to the broader lifecycle.
- **Calculations:** Implemented advanced DAX formulas and Power Query transformations to automate rolling trends and dynamic metrics.

## 📈 Business Impact & Achievements
- Successfully reduced manual reporting time by 45%.
- Established a single source of truth that accelerated cross-functional decision-making.
